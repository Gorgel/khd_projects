1. Essentials:

First order of business is to get a user account. This can be granted from any of the webmasters. Send an email asking for a user account to:

  • simon@kidshackday.com
  • ...

Once you have received your user account you can login to the KHD project page and start adding instructions for your projects.

2. Adding a project:

To add a project first login:

Then you should see something like this:

Select "add project" and you should be taken to the add project page:

In this page fill in the fields and then push the "add project"-button.

3. Tips:

  • I would highly recommend a live markdown editor when you write you project instructions. When writing in such an editor you can see your changes in real time, which helps to get an idea of how the article will look. Once you are finished with your article copy the code and then past it in the article box. An editor that can be used directly in your browser can be found HERE.

  • You can find the documentation for Markdown HERE

  • A interactive Markdown tutorial can be found HERE

  • When you embed things in your article you can also use HTML.
